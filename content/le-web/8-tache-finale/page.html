Nom du personnage

N'oubliez pas de structurer la page avec des sous-titres.

Seule le code HTML sera évalué, pas le contenu. Donc exceptionnellement, vous avez le droit de faire un copier-coller depuis Wikipédia ou tout autre site web, pour gagner du temps.
