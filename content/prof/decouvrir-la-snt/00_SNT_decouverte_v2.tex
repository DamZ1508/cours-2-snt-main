\subsection{Localisation}
\begin{enumerate}
	\item Sur votre smartphone, êtes-vous capable de lister exactement les applications qui accèdent à votre localisation ?
	\item La localisation s'effectue (entre autres) grâce \textbf{au GPS} :

			\item Que signifie le sigle GPS ?
			\item Le GPS a été inialement conçu par l'armée :

				\item russe
				\item américaine
				\item française
