\begin{titre}{2NDE}{Seconde - SNT}
\centering
{\Large \textsc{\textbf{Thème : Le web --- Algorithme PageRank}}}
\end{titre}

\textit{Lorsqu'une personne fait une requête sur un moteur de recherche, celui-ci affiche une liste de pages correspondantes, en présentant les pages les plus populaires en premier. Comment fait ce moteur de recherche pour déterminer les pages les plus populaires ?}

\textit{Dans cette activité, nous allons présenter une version simplifiée de l'algorithme PageRank, inventé par Larry Page (confondateur de Google).}

\vspace*{-0.4cm}

\section*{Présentation de l'algorithme}

La plupart des pages web présentent des liens vers d'autres pages web, appelés liens hypertextes. Le principe général est que si une page web est beaucoup référencée par les autres pages web, cette page est plus importante, et sera bien classée par les moteurs de recherche (ceci sera affiné plus tard).

Pour mesurer cette popularité par un nombre, on applique l'algorithme suivant :
\begin{itemize}
  \item on part d'une page web au hasard (on suppose que cela n'a pas d'importance, ce qui est partiellement vrai) ;
  \item sur chaque page visitée, on clique sur un lien au hasard sur cette page (pour visiter une nouvelle page) ;
    \item on répète cette étape un grand nombre de fois.
\end{itemize}
Durant l'exécution de l'algorithme, on n'oublie pas de compter le nombre de fois que chaque page a été visitée. Le score de chaque page est alors la fréquence du nombre de visites de cette page. Les pages avec le plus haut score seront visitées plus souvent que les autres.
