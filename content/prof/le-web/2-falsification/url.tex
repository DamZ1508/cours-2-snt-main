\begin{titre}{2NDE}{Seconde - SNT}
\centering
{\Large \textsc{\textbf{Thème : Le web --- Adresse URL}}}
\end{titre}

\begin{definition}
URL signifie \textbf{Uniform Resource Locator} soit en français, \og localisateur uniforme de ressource \fg{}.

C'est l'adresse d'une ressource donnée, unique sur le Web qui permet d'identifier une ressource comme une page web, un fichier pdf, une image...
\end{definition}

\begin{multicols}{2}
\begin{exemple}
Les URL sont utilisées notamment dans :
\begin{itemize}
	\item la barre d'adresse des navigateurs
	\item les liens hypertextes dans les pages web
	\item les marques-pages des navigateurs
	\item les cartes de visites, publicités, QR-code
\end{itemize}
\end{exemple}

\columnbreak

\begin{remarque}
La ressource représentée par l'URL et l'URL elle-même sont gérées par un \textbf{serveur web}.
\end{remarque}
\end{multicols}

\section*{Découpage d'une URL}

\begin{center}
\includegraphics[scale=0.5]{url.eps} 
\end{center}

\begin{enumerate}
	\item \textbf{\large \red{http://}} correspond au protocole. Cela indique au navigateur le protocole qui doit être utilisé pour récupérer le contenu. Généralement, ce protocole sera \textbf{HTTP} ou sa version sécurisée \textbf{HTTPS}.

Le Web fonctionne autour de ces deux protocoles mais le navigateur peut parfois gérer d'autres protocoles comme \textbf{mailto} qui permet d'ouvrir un client de messagerie électronique ou \textbf{ftp:} qui permet de transférer des fichiers.

	\item \textbf{\large \color{deepgreen}{www.unsite.fr}} correspond au nom de domaine. Il indique le serveur web auquel le navigateur s'adresse pour échanger le contenu. 
	
%	¿ la place du nom de domaine, on peut utiliser une adresse IP, ce qui est moins pratique et donc moins utilisÈ sur le Web.

\begin{remarque}
Dans les URL on ne peut pas utiliser n'importe quel caractère. La norme RFC1378 définit les 3 possibilités :
\begin{enumerate}
	\item des caractères alphanumériques : 0 $\rightarrow$ 9, a $\rightarrow$ z, A $\rightarrow$ Z ;
	\item des caractères spéciaux : \$ - \_ . + ! * ( ) ;
	\item des caractères encodés : \%C3\%A0 pour à, \%C3\%A9 pour é, \dots
\end{enumerate}

Concernant les caractères encodés, ils sont construits en récupérant leurs codes UTF-8 en hexadécimal (base 16), puis en préfixant chaque octet d'un caractère \% (voir 1\up{ère} Spécialité NSI).
\end{remarque}
