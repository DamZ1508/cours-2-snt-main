\section*{Etude de documents ---  Bilan}

\begin{enumerate}
	\item \emph{J'ai envie d'imprimer une image trouvée sur le site web de l'auteur·e pour l'afficher dans ma chambre.}
\begin{enumerate}
	\item \emph{En ai-je le droit ?}

		Oui. D'après l'article L122-5 du Code de la propriété intellectuelle, \enquote{lorsque l'\oe uvre a été divulguée, l'auteur ne peut interdire \ellipse{} les copies ou reproductions réalisées à partir d'une source licite et strictement réservées à l'usage privé du copiste et non destinées à une utilisation collective}.

		Donc puisque :
		\begin{itemize}
			\item l'\oe uvre a été publiée (je n'ai pas été la voler dans l'atelier secret le d'auteur) ;
			\item la source est licite (je n'ai pas téléchargé illégalement l'\oe uvre) ;
			\item l'utilisation se fait dans le cadre privé ;
		\end{itemize}
		alors j'ai le droit de l'afficher chez moi, dans mon agenda, en fond d'écran de mon téléphone, etc.
	\item \emph{La loi autorise-t-elle l'auteur·e à autoriser ou interdire cet usage ?}

		L'auteur ne peut pas l'interdire. J'ai le droit de faire un usage privé de l'\oe uvre, même si l'auteur·e n'est pas d'accord : l'article cité précédemment précise que \enquote{l'auteur ne peut interdire} les utilisations privées.
\end{enumerate}
\item \emph{J'ai envie d'utiliser une image trouvée sur le site web de l'auteur·e pour illustrer un article du journal du lycée.}
\begin{enumerate}
	\item \emph{En ai-je le droit ?}

		A priori, non. L'article L122-4 du Code de la propriété intellectuelle précise que \enquote{toute représentation ou reproduction intégrale ou partielle faite sans le consentement de l'auteur \ellipse{} est illicite}. Donc si l'auteur n'a pas donné son accord, c'est interdit.

		Étudions plus précisément les trois exemples.
		\begin{description}
			\item[Laurel] Ses mentions légales ne disent rien de plus que \enquote{\textcopyright{} 2003-2019 Laurel}. Elle n'a pas donné son accord, je n'ai pas le droit d'utiliser ses dessins.
			\item[Manu Larcenet] Il dit sur son blog que \enquote{la seule \enquote{règle} [qu'il demande] de respecter [est] de ne pas copier les images pour aller les mettre ailleurs}. Donc je n'ai pas de droit d'utiliser ses dessins.
			\item[David Revoy] Le document explique que ses dessins \enquote{ont été placés depuis l'origine sous la licence Creative Commons BY (Paternité)}. Dans le texte de cette licence, on trouve que \enquote{
					le Donneur de licence [David Revoy] Vous autorise à exercer pour le monde entier, à titre gratuit \ellipse{} les Droits accordés par la licence afin de \ellipse{} reproduire et Partager l'\oe uvre sous licence, en tout ou partie}. Donc David Revoy autorise le monde entier à réutiliser ses \oe uvres : nous pouvons l'utiliser dans le journal de l'école, et ailleurs.
		\end{description}
	\item \emph{La loi autorise-t-elle l'auteur·e à autoriser ou interdire cet usage ?}

		Oui : d'après la loi, l'autorisation de représentation et reproduction de l'\oe uvre se fait selon le consentement de l'auteur, qui peut autoriser ou interdire cet usage.
\end{enumerate}
\item \emph{Rémunération}
\begin{enumerate}
	\item \emph{Les lecteurs payent-ils pour voir les dessins publiés sur le site web ?}

		Dans les trois exemples, les dessins sont lisibles librement, intégralement, gratuitement, sans inscription.
	\item \emph{L'auteur·e gagne-t-il·le de l'argent avec la publication des dessins sur son site web ?}

		Dans aucun des trois cas les auteure et auteurs ne gagnent d'argent par la publication de dessins sur son site web :
		\begin{description}
			\item[Manu Larcenet] gagne de l'argent en vendant des BD de manière classique, sans lien avec son site web ;
			\item[Laurel] gagne de l'argent en vendant ses BD pourtant librement accessibles sur son site web (et pour différentes raisons, des gens les achètent quand même) ;
			\item[David Revoy] gagne de l'argent par les dons : beaucoup de gens donnent quelques euros par mois pour soutenir cet artiste, et comme ils sont nombreux, cela fait une somme finale conséquente.
		\end{description}
\end{enumerate}
\end{enumerate}
